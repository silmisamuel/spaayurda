MakeAndManage

MakeAndmanage is an android based application that serves as a platform for the user to develop his/her own mobile application. It resembles a content management system (CMS). This system have features like dynamic page, templates, gallery etc. The user should be provided with an account, thereby he can enter his contents to the platform and also can add the subscribers for his application. Each subscriber can download the mobile application and can view the contents. Also the subscribers can send messages among the group.

Gurukulam

Gurukulam is a project that is developed for the Kottayam District Panchayat. It is an education system tool to facilitate and ease communication between faculty, parents and students. Gurukulam forms one personally structured platform for schools or colleges. Gurukulam project revolves around one main objective which is for the up-liftment of students through the interaction between parents and teachers.
